## Use Cases

**Progress Measurement:**
- Use `assessment.md` for user self-evaluation
- 11 markers track across: awareness, reactivity, attention, discomfort tolerance, mind-wandering, relationships, values alignment, resilience, spontaneous application, decreased self-centeredness, meaning

**Adjust Practice:**
- If Aware Presence challenging → focused attention meditation
- If Radical Acceptance difficult → RAIN practice
- If Sphere Sovereignty not guiding behavior → journaling about control
- If Virtuous Response not automatic → virtue contemplation
- If Interconnected Living needs work → loving-kindness meditation

**Further Learning:**
- Classical sources for philosophical depth
- Neuroscience for scientific validation
- Community resources for ongoing support
