Visual Production Notes
Image Requirements
App Screenshots:

Export at 2x resolution for retina
Include device frames (use Facebook Design frames)
Consistent shadow treatment (30% opacity, 20px blur)
Status bar time always shows 9:41 AM (Apple standard)

Website Graphics:

SVG for all icons and illustrations
WebP for any photographic content (none planned)
Gradient meshes as CSS only (performance)
Lazy loading for below-fold images

Typography Specifications
Font Stack:

Primary: SF Pro Display (Apple), fallback to system
Weights needed: 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
Size scale: 12, 14, 16, 18, 20, 24, 28, 32, 36, 48, 72

Text Treatments:

Headlines: -0.02em letter spacing
Body text: 1.5 line height
Captions: 0.02em letter spacing
CTAs: 0.01em letter spacing, always sentence case

Animation Specifications
Micro-interactions:

Button hover: translateY(-2px), 0.2s ease
Card hover: shadow increase, 0.3s ease
Progress bars: width transition, 0.5s ease-out
Breathing circle: scale 1 to 1.2, 4s ease-in-out infinite

Scroll Animations (optional):

Fade up on scroll (opacity 0 to 1, translateY 20px to 0)
Stagger delays for card groups (0.1s between items)
Parallax on hero phone mockup (subtle, 0.5 rate)
